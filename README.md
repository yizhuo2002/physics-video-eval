# Human Evaluation Data: Phys-FramePackGRPO

This repository contains the human evaluation dataset for the Master of Computing Capstone Project: **Enforcing Newtonian Invariants in Image-to-Video Generation via Critic-Free Reinforcement Learning** (National University of Singapore).

This data supports the human evaluation section (Section 5.6) of the capstone report, validating that the automated physical reward ($S_{phys}$) improvements translate to human-perceivable gains in physical realism in generated videos.
